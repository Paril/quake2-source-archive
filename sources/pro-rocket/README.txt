PRO-ROCKET 1.0
-------------------------------------------
For more information (and source code) -
http://www.planetquake.com/pro-rocket


To install PRO-ROCKET
-------------------------------------------

	1. Unzip this archive (with pathnames) into your quake2\ directory.


To run PRO-ROCKET
-------------------------------------------
	Pro-rocket is a weapons and physics tweak for Quake2.

	Pro-rocket requires no additional commands or config settings,
	it operates like a normal server.

	example start-up string :
	c:\quake2\quake2 +set game pro-rocket

Have fun,
