PRO-CTF 1.0
-------------------------------------------
Zoid's Capture the Flag mod with Pro-Rocket 1.0 support.

	For more information (and source code) -
	http://www.planetquake.com/pro-rocket



*** you must have CTF installed before installing PRO-CTF ***



To install PRO-CTF
-------------------------------------------
	1. Unzip this archive (with pathnames) into your QUAKE2 directory

	2. copy the pak files (pak0.pak etc..) and server.cfg files
	from quake2\ctf to quake2\pro-ctf

To run PRO-CTF
-------------------------------------------
	Pro-CTF requires no additional commands or config settings,
	it operates like a normal CTF server.

	example start-up string :
	c:\quake2\quake2 +set game pro-ctf +exec server.cfg



Have fun
