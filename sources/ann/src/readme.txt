This source code package is being prepared for release on 06 October 2000.

I enjoyed writing Annihilation, and I know you loved playing it. There's been
demand for the source code, and I have nothing to hide ( erm =] ), so here it
is.

You must agree to the conditions stated in the file SLICENSE.TXT, provided with
this package.

If you make any changes to the mod, feel free to email us and let us know how
things are going. You are also free to distribute your modification, provided
that the source code to the latest version is always freely available.

NOTE: Better use my private email now, since the telefragged hosting has
ended: tom@holodeck.f9.co.uk .

You should decompress this source code into your Annihilation directory,
maintaining the directory structure. The source code for Annihilation will
exist in the SRC folder, and the ANNPLAY folder will contain the AnnPlay
source.

A few things:
* The majority of the Annihilation modifications are in g_brrr.c (supported by
  g_brrr.h), although there are many others scattered throughout the code.
* As some code was written by people other than me, I take no responsibility
  for its readability.
* There are many variables you can tweak in g_brrr.h for "interesting" results
  :)

Good luck, happy hacking, and have fun with the source.
Stay in touch :)

-- Chawmp (used to be eLiTe)
