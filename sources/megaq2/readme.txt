MegaMan Quake2 Version 1.2
 - update to fix r1q2 crash

Also includes
 - New enhancement Buster Upgrade
 - New model for Double Buster
 - Other minor bug fixes

To install extract to your quake 2 directory. The file structure of "megaq2" already exists in the zip.
i.e.
Extract to c:\quake2
This will create the mod folder c:\quake2\megaq2

(Remember that some versions of quake 2 install to c:\program files\quake II)

Remember to bind keys to +use and drop tech
i.e.
bind mouse2 +use
bind q drop tech
